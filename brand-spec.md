# 零里说 · 无障碍在线导览 - Demo临时视觉规范

## 状态

本规范是 Figma Make 参考项目的实现记录，不是正式品牌手册。当前 Demo 不新增参考项目之外的视觉语言。

## 主画板

- iPhone 16 标准版：393×852 CSS px。
- 截图：DPR 3，1179×2556 px。
- 桌面环境保持393px宽并居中；窄于393px时贴合视口。
- 模拟系统层包括9:41、126×30灵动岛、电量与信号图形、134×5 Home Indicator。

## 本地素材

- 地图：`demo/assets/figma-make/018740dea30b01362c4a3f15d652f695.png`。
- 讲解员图标：`demo/assets/figma-make/___1.png` 至 `___15.png`，依次对应 `g1` 至 `g15`。
- 参考插画：`demo/assets/figma-make/b104cde6e08dbe970d654b0e59630fd1.png`，当前界面未调用。
- 所有17个 PNG 与参考项目 `src/imports` 的 SHA-256 一致。
- 界面图标使用本地内联 Lucide SVG，不加载网络资源。

## 字体与层级

- 界面正文：系统无衬线中文字体，16px 基准，24px 行高。
- 品牌字与地点标题：系统中文衬线回退字体。
- 地图图例：14px；说明文字：12px。
- 讲解正文：16px，26px 行高；不将重要文字烘焙进界面图片。

## 形状与阴影

- 顶部筛选和主按钮使用全圆角胶囊。
- 底部面板和弹层顶部圆角为32px。
- 讲解员使用原始透明 PNG；WC 使用圆角方形，坡板使用圆形，服务点使用45度菱形。
- 主面板阴影：`0 -8px 30px rgba(22, 51, 33, 0.08)`。
- 地图图像以 `object-fit: cover`、60%不透明度覆盖地图区域。

## 动效

- 筛选状态：300ms。
- 底部弹层：500ms ease-out。
- 定位点使用轻微呼吸动画。
- 系统启用“减少动态效果”时，将非必要动画和过渡压缩至近乎即时。

## 表述边界

- 地图固定显示“示意地图，点位与路线待现场核验”。
- 坡板固定使用“可提供坡板”或“使用方式待核验”。
- 不使用“完全无障碍”等未经现场验证的承诺。
